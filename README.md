# BMW Global Sales Performance Dashboard

## 📊 Project Overview
This Power BI dashboard provides a comprehensive analysis of BMW's global commercial performance. It tracks massive datasets, including **19.01T in total revenue** and **253M units sold** across **45 different countries**.

## 🚀 Key Performance Indicators (KPIs)
* **Total Revenue:** 19.01T.
* **Total Units Sold:** 253M.
* **Average Revenue:** 75.04K.
* **Product Range:** 11 distinct Car Models.
* **Market Reach:** 45 Countries.
